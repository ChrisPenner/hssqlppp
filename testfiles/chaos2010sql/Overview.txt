Chaos 2010

Inspired by an old Spectrum game called
[Chaos](http://en.wikipedia.org/wiki/Chaos_(video_game)).

The idea is to keep all the data in the database, and use constraints,
transactions and stored procedures to keep things consistent and
enforce the rules.

The transformed versions are an attempt to show the sql as it goes into the database after being processed with the extension system. It's supposed to show sql from the original file, greyed out if it has been changed during the transform, and the extra generated sql from the transform, and it almost works.
